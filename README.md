# HonestQQ
基于环信以及leanCloud作为后台的即時通訊項目。主要使用到的技术有：自定义控件实习好友列表，以及滑动删除会话功能；ZXing扫一扫加好友的功能实现，OKHttp，EventBus，工厂设计模式，shareSDK短信验证等
